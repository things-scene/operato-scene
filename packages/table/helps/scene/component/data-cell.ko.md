# table cell
## Properties
1. Border Style - 선택된 셀의 테두리의 굵기 및 색상을 변경. 스타일 값을 입력 후 ![Table-셀 스타일 종류][table-04] 중 적용할 셀의 버튼을 누른다.
![Table-셀 스타일][table-05]
1. Border Type - 선택 된 셀의 선의 종류를 변경. 테두리 종류를 선택 후 ![Table-셀 스타일 종류][table-04] 중 적용할 셀의 버튼을 누른다.
