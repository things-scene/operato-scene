# legend[zh]
![组件-Legend][legend-01]

设置状态值并根据所设置的状态创建类别的组件。


## properties

1. 行：决定用几个行来显示该类别。  
2. 列：决定用几个列来显示该类别。  
3. 方向：将会显示类别的方向。  
4. 圆化：确定边缘的圆度。  
5. 字段：指定数据的哪些字段值是决定状态值的字段。  
6. 默认颜色：如果不在状态范围内，就会默认显示的颜色。  
7. 状态设置：指定值的状态范围和颜色。  ![状态设置][legend-02]
